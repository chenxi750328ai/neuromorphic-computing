# Phase 3 · 小样本实验结果

> 陈正共 · 本地跑通（GitHub 暂不可达）· 2026-06-05

## 结论（先看这个）

| 方式 | 含义 | 5-shot test acc | 是否过 85% 线 |
|------|------|-----------------|--------------|
| **A. 从头训练** | 只用 50 张图，随机初始化 | **66.84%** | ✗ |
| **B. 全量预训练 + 微调** | 先 Phase2 全量 ANN，再只用 50 张微调 | **96.32%** | ✓ |

**解读：** Phase3 要验证的是「标注很少时还能不能用」。  
- 若 **从零学**，50 张图远远不够（接近瞎猜的 10% 到 66%）。  
- 若 **已有全量模型再适配新少量样本**（迁移/微调），5-shot 仍可保持 **96%+**，接近全量 98.10%。  

工业上小样本通常是 **B 路线**；A 路线需要 meta-learning 等更强方法（留 Phase3 v0.2）。
