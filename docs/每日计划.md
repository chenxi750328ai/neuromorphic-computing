## 自驱动（小脑 + hook + loop）

| 组件 | 命令 |
|------|------|
| 安装 hook | `./scripts/install-neuro-cursor-hooks.sh` |
| 小脑 tick | `python3 scripts/neuromorphic-cerebellum.py`（写 wake **并刷新看板**） |
| 看板同步 | `python3 scripts/neuro-dashboard-sync.py`（小脑已自动调用） |
| 定时 loop | `./scripts/neuro-drive-loop.sh 300`（或本会话 notify 5m） |
| Cursor `/loop` | `/loop 10m` 监听 stdout 中 `AGENT_LOOP_TICK_NEURO` |
| 项目 hook | `.cursor/hooks.json` → `scripts/neuromorphic-cursor-wake.py` |

须在 Cursor **打开类脑仓根** `/home/cx/neuromorphic-computing`；配置见 `config/neuro-cerebellum-config.json`。
