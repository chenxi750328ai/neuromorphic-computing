## 自驱动（小脑 + hook）

| 组件 | 命令 |
|------|------|
| 小脑 tick | `python3 /home/cx/vcompany/scripts/neuromorphic-cerebellum.py` |
| 定时 loop | `/home/cx/neuromorphic-computing/scripts/neuro-drive-loop.sh 600` |
| Cursor hook | 已接：`neuromorphic-computing/.cursor/hooks.json` → `~/.cursor/hooks/neuromorphic-cursor-wake.py`（**仅类脑工作区**） |
