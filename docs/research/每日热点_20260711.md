## 1. 今日要点

### 1.1 北大杨玉超团队 · Science · 相变忆阻器神经动力学芯片

北京大学集成电路学院杨玉超（Yuchao Yang）团队联合中科院上海微系统所宋志棠团队，在 *Science* 发表题为 **「A sub–10-millisecond neural dynamical system based on phase-change memristors」** 的研究。团队采用 **40 nm** 工艺流片，核心存算阵列面积约 **0.28 mm²**，运行频率 **50 MHz**，单步神经动力学迭代时延 **2.12 ms**（论文称亚 10 ms 级 NDS 硬件）。相较专用加速器提速 **3.82–36.27×**、功耗降 **11.75–24.73×**；在脑皮层表面重建任务中，端到端对比 NVIDIA A100 提速 **50.38–478.18×**。技术路径为「可控存内计算」：将相变忆阻器电导漂移与多级电导特性直接用于自适应积分步长搜索与存内乘累加，减少数据搬运。

- **来源**：[PKU 作者页](https://yzhu.io/publication/physics2026science/) · [南燕新闻网](https://news.pkusz.edu.cn/info/1003/9813.htm) · [ECNS 报道](https://www.ecns.cn/m/news/sci-tech/2026-07-06/detail-ihffyeqp1389975.shtml)（2026-07-03/06）

### 1.2 西北大学 · Nature Communications · 小脑启发 memtransistor

西北大学 Mark C. Hersam 等团队在 *Nature Communications* 发表 **「Cerebellum-inspired memtransistors enable emergent differentiation for hardware-efficient novelty detection」**（2026-07-10 上线）。团队基于不对称接触栅控 **MoS₂ memtransistor**，利用偏压极性依赖的兴奋/抑制短期可塑性，模拟小脑「常态抑制、异常触发」的突触分化机制。在 ECG 心律失常检测中，可在 **单次心跳时间尺度**（媒体报道称约 **1/5 心跳** 内）识别异常，准确率 **>98%**，运算量约为硅基方案的 **1/10,000**。

- **来源**：[Nature Communications](https://www.nature.com/articles/s41467-026-75212-4) · DOI: `10.1038/s41467-026-75212-4` · [Neuroscience News](https://neurosciencenews.com/ai-cerebellum-memtransistor-31036/)（2026-07-10）

## 3. 对 GWT / 分工 / 硬件 的启示

### 3.1 GWT

两篇顶刊的共同信号是：**生物启发 ≠ 更大 GPU batch**，而是把「何时计算」（小脑：常态跳过/异常触发）和「在哪计算」（忆阻器：存内连续动力学）做到物理层。对 GWT 的直接启示是：M1–M4 之间的竞争接入不应仅是 top-k 分数排序，可探索 **「低活动默认抑制 + 高显著性才写入 workspace」** 的门控策略（软件仿真即可验证），与西北大学 emergent differentiation 逻辑同构，但 **不要求** 复现其器件。北大 NDS 芯片侧证的是：若未来 M1 感知模块要跑 **连续流形/表面类** 任务，毫秒级硬件迭代有外部可行路径——这与当前 MNIST 25-tick 分类基座是 **不同阶段问题**，现网仍按 `GWT_愿景与阶段依赖_V1.md` 走 Phase5 N-MNIST 再谈模块异构。

### 3.2 分工（Phase4.1）

Phase4.1 bench 已证明：**主瓶颈是跨板通信与 SSH 编排（comm_ratio≈99.9%）**，Atlas NPU 算力本身稳定 ~3.4 ms。北大/西北大学成果 **不改变** 这一结论——它们解决的是 **器件级存内动力学** 或 **边缘新奇检测**，不是 WSL↔Atlas payload 协议。分工 v1（Atlas 常驻 daemon）仍成立；今日热点 **不支持** 回头做「每帧 sshpass 转发器」，也 **不支持** 以忆阻器新闻为由提前启动分板 GWT（文件明确：Phase4.1 关单前不做分板 GWT）。

### 3.3 硬件

硬件栈应保持 **诚实双轨**：**近端** Atlas 200DK + PYNQ-Z2（CHARTER 边缘/FPGA）把 MNIST→N-MNIST 跑通；**远端观察** 相变忆阻器 NDS 与 2D memtransistor 作为 **2030+ 可选加速器类别**，而非替代昇腾 OM 或 FPGA HLS 路径。北大芯片 40 nm / 0.28 mm² 说明 **存内计算可在成熟工艺落地**，对 CHARTER「边缘低能耗」有方向性鼓舞，但 **无现成 IP、无驱动、无与 snnTorch 导出链对接**——PL 应抵制「换硬件叙事」压力，先把 daemon bench N≥100 分布关单。
